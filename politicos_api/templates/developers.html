Quer contribuir?

A base da Politicos API é o framework Tornado. Os dados são salvos e indexados no ElasticSearch e também existe uma camada de cache utilizando o Redis.

Arquitetura

O projeto foi dividido em dois módulos: o coletor e a API.

Coletor

Pega os dados no site do TSE e salva no ElasticSearch. Criamos um índice para cada ano e um alias para unificar todos os índices.

API

Faz a consulta no ElasticSearch e armazena temporariamente no Redis.

Informações

O endereço da API é:

https://politicos.olhoneles.org/api/v1/

Você pode ver todas as rotas da API no item Rotas da API no menu superior.

Alguns exemplos de rotas e filtros:

https://politicos.olhoneles.org/api/v1/educations/
https://politicos.olhoneles.org/api/v1/candidacies/?ano_eleicao=2014

Software Livre

O código da API é livre e licenciado sob a Affero GPL. Você pode pegar o código executando o seguinte comando:

$ git clone https://github.com/olhoneles/politicos.git

Contribuindo

Vamos deixar as contribuições para este projeto tão fácil e transparente quanto possível. Portanto faça um fork do repositório e nos envie suas melhorias.

Usamos o GitHub para armazenar o código, acompanhar as questões e solicitações de recursos.

Ajuda

Você pode abrir um ticket via GitHub, mas se preferir temos uma lista de email do projeto e também estamos online via IRC no canal #olhoneles na Freenode.
